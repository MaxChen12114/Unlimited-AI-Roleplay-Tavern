Settings

本地数据保存在浏览器;启用云同步后也会上传到你的 Cloudflare KV

🔒 密码保护（Auth）

云同步永远需要密码（用 Cloudflare Secret CHAT_PASSWORD 鉴权）。 聊天密码保护可选:开启后调用 /api/chat 需带 token,关闭后聊天接口公开(任何扫到域名的人都能用你的余额)。

☁️ 云同步（Cloud Sync）

把全部数据(角色卡 · 道具 · preset · 亲密度 · 费用日志 · 对话历史)同步到 Cloudflare KV,换浏览器换设备都能看到。 开启后本地改动会延迟 3 秒自动推送。不同步:密码 token / 同步状态本身。

外观风格(Visual Theme)

切换桌面 UI 视觉风格。极简 = 深色平面、低干扰、阅读优先;毛玻璃 = 亮色渐变 + 半透模糊。 蜜桃 / 少女 = 暖色双主题,带呼吸脉动效果(启动音频需点击页面一次解锁)。设置实时生效并保存到本地。

💡 蜜桃/少女主题需要 /assets/lewd-ambient.mp3(自配 CC0 心跳/呼吸 loop,仓库不打包版权资源)。无音频时走 CSS 慢呼吸 fallback。Shift+Esc 可紧急切回极简。

🎵 主题音效(Ambient Audio)

控制 蜜桃 / 少女 主题下的环境音和呼吸脉动。关闭后即使切到这两个主题也不会发声、不脉动(背景仍是配色)。其他主题不受影响。设置仅本设备生效(不进云同步)。

70%
💡 Shift+Esc 是全局紧急停止快捷键,按下后自动切回极简主题。首次启动需点击页面任意位置一次解锁浏览器音频权限。

提示词预设库(Prompt Presets)

解限提示词 PROMPT_1/2/3 之后追加的语调约束层。启用的预设按顺序以空行拼接注入。 红线:只追加,不替换解限 base。名为 COT 的预设在快速模式(DeepSeek)下自动禁用,避免与 native thinking 冲突。

费用日志(Cost Log)

仅记录快速模式(DeepSeek)实际计费。独立于本地历史——清除本地历史不会清掉这里。按日累加并永久保留,存 localStorage.cfw_cost_log_v1。

本地对话记忆(Local History)

开启后:刷新页面/重开浏览器仍会恢复对话。关闭后:仅本次页面会话有效,不再落盘保存。
⚠️ 清除历史只会删除本地对话,不会影响“网页自定义人物模板”。

网页自定义人物模板(Web Custom Persona)

⚠️ 仅当右上角为 😇 且未选择角色卡时生效;😈 模式与已选角色卡时此处都会被忽略。
高级用法:可在此填写任意 system prompt,包括你自己写的解限内容——系统不限制内容,效果取决于你写什么。
清除模板只会删除网页自定义内容,不会影响本地历史或角色卡。
